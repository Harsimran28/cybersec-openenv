# 🛡️ CyberSecurity RL Agent (OpenEnv Style)

## 🚀 Overview
This project simulates a cybersecurity defense system using Reinforcement Learning (RL).

An AI agent learns to:
- Detect threats
- Protect system health
- Take optimal defensive actions

---

## 🎯 Features
- Custom RL Environment (Cybersecurity simulation)
- DQN Agent using PyTorch
- Experience Replay + Epsilon Decay
- Task-based design (increasing difficulty)
- Automated Grader System
- Streamlit UI for live demo

---

## ⚙️ Tech Stack
- PyTorch
- NumPy
- Streamlit
- Matplotlib
